# RecipeIngredientConverter
Recipe Ingredient Converter is a web application that helps users adjust ingredient quantities across different units and servings. Enter an ingredient, quantity, unit, and number of servings, and the app calculates the equivalent quantities in other units.
## Features
- Convert ingredient quantities between cups, grams, ounces, and teaspoons.
- Adjust quantities according to the number of servings.
- Display results in a clear, easy-to-read list.
- Responsive design suitable for desktop and mobile.

## Technologies Used
- **HTML** – Structure of the application.
- **CSS** – Styling and layout.
- **JavaScript** – Conversion logic and dynamic DOM updates.

## Usage
1. Open `index.html` in a web browser.
2. Enter an ingredient name and quantity.
3. Select the unit and number of servings.
4. Click **Convert** to view equivalent quantities in other units.
